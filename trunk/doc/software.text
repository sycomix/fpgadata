## Packages installed
----------------------------------------------------------------------------
These additional packages are needed to run some of the tools developed here.

Packages:
    r-base-core (3.1.1-1+deb8u1)	R language
    perl-tk (1:804.032-3+b1)		PerlTk graphics


Helpful commands:
    Resynchronize the package index files from their sources.
    % sudo apt-get update

    Inspect package
    % apt-cache show  <pkg>

    Install the package
    % sudo apt-get install  <pkg>
